# 🚀 SJSU Deep Learning – Execute the Colabs & Create Video Explanations

This repository contains executed Google Colab notebooks and recorded video explanations for a complete deep learning journey — from mathematical foundations to advanced architectures across multiple frameworks.

The goal of this project was to:
- Execute each provided Colab notebook
- Understand the core concepts
- Record clear video explanations
- Demonstrate both theoretical understanding and practical implementation

---

# 📚 Full Deep Learning Roadmap Covered

This project walks through deep learning in a structured progression:

1. Mathematical Foundations  
2. NumPy (From Scratch)  
3. PyTorch (Primitives → Production)  
4. TensorFlow & Keras (Primitives → Production)  
5. JAX & Flax (Functional Deep Learning)  
6. Advanced Architectures (ResNet, Attention, Transformers)
